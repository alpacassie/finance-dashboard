# Finance Dashboard

A personal finance dashboard built with Next.js and Supabase.

## Live Demo

**[https://finance-dashboard-tau-ecru.vercel.app](https://finance-dashboard-tau-ecru.vercel.app)**

## Features

- Date filtering (week, month, year, all time)
- Spending by category chart
- Monthly trends visualization
- Transaction list with multi-select filters
- Recurring expenses tracker
- Income vs spending views

## Tech Stack

- Next.js 16
- Supabase
- Recharts
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```
